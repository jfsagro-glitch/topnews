# Документация: Кнопка "Возобновить работу"

## Описание

Добавлена новая кнопка **"▶️ Возобновить работу"** в админ-панель бота для песочницы (sandbox). Кнопка позволяет:
- Снять глобальную остановку системы
- Запустить сбор новостей 
- Получить модальное уведомление о статусе

## Расположение

Кнопка отображается в главной админ-панели управления ботом:
- **Путь**: `/admin` → Главная панель управления
- **Видимость**: Только когда система остановлена (`get_global_stop() == True`)
- **Доступность**: Только в окружении `sandbox` (APP_ENV=sandbox)

## Как работает

### 1. Отображение кнопки

Кнопка добавляется в метод `_build_sandbox_admin_keyboard()` (bot.py:413-426):

```python
# Добавляем кнопку "Возобновить работу" только когда система остановлена
if is_stopped:
    keyboard.append([InlineKeyboardButton("▶️ Возобновить работу", callback_data="mgmt:resume_work")])
```

### 2. Обработка нажатия

Callback-обработчик `mgmt:resume_work` (bot.py:~1750):

1. **Проверка доступа** - только администраторы
2. **Проверка окружения** - только sandbox  
3. **Снятие остановки** - вызов `set_global_stop(enabled=False)`
4. **Модальное уведомление** - `show_alert=True`
5. **Запуск сбора** - асинхронная задача `_trigger_news_collection()`
6. **Обновление UI** - кнопка исчезает, статус обновляется

### 3. Запуск сбора новостей

Метод `_trigger_news_collection()` (bot.py:~4299):

```python
async def _trigger_news_collection(self):
    """Запускает сбор новостей в песочнице (фоновая задача)"""
    try:
        logger.info("[ADMIN] Triggering news collection in sandbox...")
        if self.collector:
            results = await self.collector.collect_all(force=True)
            logger.info(f"[ADMIN] Collection completed: {len(results)} items collected")
    except Exception as e:
        logger.error(f"[ADMIN] Error during news collection: {e}", exc_info=True)
```
